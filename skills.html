Skills

These are the web design skills we learned and used in this final project.

HTML5

We used semantic HTML tags such as header, nav, main, section, article, and footer.

CSS3

We used CSS to design colors, spacing, typography, buttons, cards, and page layouts.

Flexbox

Flexbox was used for the navigation menu, team cards, and buttons.

CSS Grid

CSS Grid was used to create responsive cards and the project gallery layout.

Responsive Design

The website changes layout for smaller screens such as phones and tablets.

Contact Form UI

We created a contact form design using input fields, a textarea, and a submit button.
