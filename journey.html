Learning Journey

This page explains how we improved our HTML and CSS skills during the semester.

How We Built This Website

Our team created this portfolio step by step. We started with simple HTML structure, then added CSS design, layouts, images, responsive design, and finally prepared the website for GitHub Pages deployment.

1

Planning the Website

We decided the pages, content, colors, and team roles before writing the code.

2

Writing HTML Structure

We created multiple pages using semantic tags such as header, nav, main, section, article, and footer.

3

Designing with CSS

We styled the website using colors, spacing, typography, cards, buttons, and shadows.

4

Using Flexbox and Grid

Flexbox helped us align navigation and team cards, while CSS Grid helped us create gallery layouts.

5

Making It Responsive

We used media queries so the website works better on phones, tablets, and desktop screens.

What We Learned

We learned how to organize files, connect pages, use images, create layouts, and build a complete website using only HTML5 and CSS3.

Most Useful Skill

The most useful skill was learning responsive design because modern websites should look good on different screen sizes.

Teamwork

We divided the work, checked each page, and kept the design consistent across the website.
